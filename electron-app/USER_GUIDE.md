## Installation Guide for Windows Users

### System Requirements
- **Windows 10 or 11** (64-bit recommended)
- **Windows 8 or 8.1** (32-bit supported)
- **4 GB RAM** minimum (8 GB recommended)
- **Internet connection**
- **Webcam and microphone** (for video calls)

## Installing the Application

### Method 1: Installer Version (Recommended)

1. **Download** the installer:
   - For 64-bit Windows: `Almajd Meet Setup 1.0.0.exe`
   - For 32-bit Windows: `Almajd Meet Setup 1.0.0 ia32.exe`

2. **Run the installer**:
   - Double-click the downloaded file
   - Windows might show a security warning - click **"More info"** then **"Run anyway"**

3. **Follow installation steps**:
   - Choose installation location (or keep default)
   - Select **"Create desktop shortcut"**
   - Click **"Install"**

4. **Launch the app**:
   - Double-click the desktop shortcut
   - Or find it in Start Menu ‚Üí "Almajd Meet"

### Method 2: Portable Version (No Installation)

1. **Download** the portable version:
   - For 64-bit: `Almajd-Meet-1.0.0-x64-portable.exe`
   - For 32-bit: `Almajd-Meet-1.0.0-ia32-portable.exe`

2. **Move to desired location**:
   - Desktop, Documents, or USB drive

3. **Run the application**:
   - Double-click the .exe file
   - No installation needed!

## Using Almajd Meet Desktop

### First Time Setup

1. **Grant Permissions**:
   - When prompted, allow access to:
     - Camera
     - Microphone
     - Screen sharing (if needed)

2. **Entry Screen**:
   You'll see a welcome screen with:
   - **Room ID field**: Enter the meeting room number
   - **Your Name field** (optional): Enter your name
   - **JOIN MEETING button**: Click to start

### Joining a Meeting

1. **Get Room ID**:
   - Your host will provide a room number (e.g., 1, 2, 123)

2. **Enter Room ID**:
   - Type the room number in the "Room ID" field

3. **Enter Your Name** (optional):
   - Type your name (or leave blank for "Host")

4. **Click "JOIN MEETING"**:
   - Wait for the app to connect
   - You'll enter the meeting room

### During a Meeting

#### Video Controls
- **Camera button**: Turn camera on/off
- **Microphone button**: Mute/unmute
- **Screen Share button**: Share your screen
- **Settings**: Adjust camera/microphone settings

#### Meeting Controls
- **Chat button**: Send text messages
- **Participants**: See who's in the meeting
- **Whiteboard**: Draw and share (host mode)
- **End Meeting**: Leave or end the meeting

#### Screen Sharing
1. Click the **Screen Share** button
2. Your entire screen will be shared automatically
3. Click **Stop Sharing** to stop

### Leaving a Meeting

1. **Click "Leave Meeting"** button
   - Located in the meeting controls

2. **Auto-Return**:
   - You'll automatically return to the entry screen
   - Your Room ID will be saved for easy rejoining!

3. **Rejoin**:
   - Just click **"JOIN MEETING"** again
   - No need to re-enter the Room ID

## Troubleshooting

### Camera/Microphone Not Working

**Solution:**
1. Close the app completely
2. Reopen Almajd Meet
3. Grant permissions when prompted
4. Check Windows Privacy Settings:
   - Go to **Settings** ‚Üí **Privacy**
   - Enable **Camera** and **Microphone** for apps

### Screen Sharing Not Working

**Solution:**
1. Make sure you're using the desktop app (not web browser)
2. Grant screen recording permissions when prompted
3. Restart the app if needed

### Cannot Join Room

**Possible Reasons:**
- ‚ùå Room ID is incorrect
- ‚ùå Room doesn't exist yet
- ‚ùå No internet connection

**Solution:**
1. Check the Room ID with your meeting host
2. Verify internet connection
3. Contact support if problem persists

### App Won't Start

**Solution:**
1. Restart your computer
2. Re-install the application
3. Check if antivirus is blocking it
4. Make sure Windows is up to date

### "Unknown Publisher" Warning

**This is normal!**
- Windows shows this for apps without code signing
- **It's safe to use**
- Click **"More info"** ‚Üí **"Run anyway"**
